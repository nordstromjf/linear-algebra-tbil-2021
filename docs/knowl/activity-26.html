Activity 2.1.10.

Consider the set \(V=\setBuilder{(x,y)}{y=e^x}\) with operations defined by

\begin{equation*} (x_1,y_1)\oplus (x_2,y_2)=(x_1+x_2,y_1y_2) \hspace{3em} c\odot (x_1,y_1)=(cx_1,y_1^c)\text{.} \end{equation*}

Let \(a, b\) be unspecified scalars and \(\vec u \in V\) be the same vector as in the previous exercise.

(a)
\begin{equation*} \text{Compute } (a+b)\odot \vec u \end{equation*}
(b)
\begin{equation*} \text{Compute } \left(a\odot \vec u\right)\oplus \left(b\odot \vec u\right). \end{equation*}
