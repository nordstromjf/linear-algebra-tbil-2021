Think about the answers to the following questions:

  1. Are all of the vectors in \(S\) also in \(\IR^3\text{?}\)
  2. For any unspecified \(\vec{u}, \vec{v} \in S\) is it the case that \(\vec{u} + \vec{v} \in S\text{?}\)
  3. Let \(\vec{z}\) be the additive identity in \(\IR^3\text{.}\) Is \(\vec{z} \in S\text{?}\)
  4. Are there any properties of a vector space you think might not apply to \(S\text{?}\) If so, check these.
